# 변경 이력 (Changelog)

이 파일은 [Keep a Changelog](https://keepachangelog.com/ko/1.1.0/) 형식을 따르며,

## [1.0] — 2026-07-20

첫 출시 🎉 — **종이책의 문장을 사진 한 장으로 소장하는 앱, 밑줄.**

### 추가 (Added)
- **표지 촬영으로 책 등록** — 온디바이스 OCR로 제목·저자 자동 인식,
  뒤표지 바코드(EAN-13)를 찍으면 ISBN까지 자동 입력
- **손가락 형광펜 문장 추출** — 페이지를 찍고 원하는 문장 위를 손가락으로
  긋기만 하면 그 문장만 골라 스크랩 (지우개·전체 선택·전체 지우기 지원)
- **페이지 번호 자동 감지** — 페이지 사진 가장자리 숫자를 휴리스틱으로 추정
- **서재 & 문장 보관함** — 책별 표지 그리드, 전체 문장 검색, 즐겨찾기
- **오늘의 문장** — 매일 스크랩 하나를 다시 만나는 회고 화면
- **홈 화면 위젯** — 오늘의 문장 (small / medium / large)
- **잠금화면 위젯** — 사각형 / 인라인 / 원형
- **이미지로 공유** — 문장을 페이퍼 감성 카드 이미지로 만들어 SNS에 공유
- **iCloud 동기화** — SwiftData + CloudKit 기기 간 자동 동기화 (실패 시 로컬 폴백)
- **다크 모드 & 앱 아이콘 3종** — 라이트 / 다크 / 틴티드 대응

### 특징
- OCR은 100% 온디바이스(Vision)로 동작 — 사진이 외부로 나가지 않고 오프라인에서도 사용 가능
